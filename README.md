# Saga - TechSolutions Webao Help 3.0

An AI-powered knowledge base chatbot for TechSolutions support team.

## Features

### AI Chatbot
- Ask questions in natural language
- Get instant answers from the knowledge base
- Voice recognition support
- Related topics suggestions

### Knowledge Base Management
- Add, edit, and delete articles
- Search and filter content
- Bulk import via JSON
- Category and tag organization

### Dashboard
- User profile and statistics
- Quick actions menu
- Team status tracking
- Knowledge base analytics

## Quick Start

1. Configure Supabase credentials in `js/supabase-config.js`
2. Open `index.html` to log in
3. Navigate to `chat.html` to use the AI assistant
4. Use `knowledge-admin.html` to manage content
